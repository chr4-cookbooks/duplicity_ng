## Usage

### duplicity_ng::default

Include `duplicity_ng` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[duplicity_ng::default]"
  ]
}
```

5. Run the tests, ensuring they all pass
6. Submit a Pull Request
